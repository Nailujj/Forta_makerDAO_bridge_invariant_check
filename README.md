# MakerDAO's Bridge Invariant Check Bot

## Description

This bot detects when the following MakerDAO's L1/L2 bridge invariant is violated : 
```
L1Dai.balanceOf(Escrow) >= L2Dai.totalSupply()
```

## Supported Chains

- Ethereum
- Optimism
- Arbitrum
